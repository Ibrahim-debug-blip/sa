## Shoaib Akhtar ##
This HTML website is created on the biography of Shoaib Akhater. His complete cricket journey is covered in this website including his domestic career.
## Features ##
Alot of work is done on Style.css but within the html file.
Buttons and Containers are used as well in order to seperate different types of stats.
Color contrasts are used as well.
Different format is discussed in seperate pages.
